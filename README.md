# Student Leave Tracker System
The Student Leave Tracker System is a web-based application developed to manage and monitor student leave requests in a simple and structured way.  
It reduces manual paperwork and also provides useful insights through data analysis and visualizations, which helps institutions make better decisions.

This project was developed as part of an academic project and also focuses on applying data analysis concepts on real-world data.

## Data Analysis & Visualization

This project includes **basic data analysis and visualization**, which adds value for data-related roles.
